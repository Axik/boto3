Welcome to boto3
================

Boto3 is a high-level interface to a growing number of Amazon Web
Services.


Requirements
------------

* Python 3.3+ first, but also works on Python 2.6+
* botocore==0.24.0
* six>=1.4.0
* jmespath==0.1.0
* python-dateutil>=2.1
